README file for SudokuSolver

The input file name is "Puzzle.txt" 
Puzzle.txt is inputed into an array and solved using a pack tracking algorithm which uses 
A linked list and a stack.

User is prompted to how many steps they want the puzzle to be solved in and is shown the solved puzzle as well as how many steps are remaining on the command line.

To run the file type in ./guerrier
